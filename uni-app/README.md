# 快速启动
1. 修改服务端接口地址 分别修改utils包下domain.js中url和wsConfig.js中WS 的值 修改为自己的服务端接口地址 <br>
2. 拉下代码后使用HBuilderX 打开，运行-运行到手机或模拟器-运行到Android App基座<br>
